# Serial Monitor v1.1.6

출시일: 2026-07-21

## COM 포트와 연결 안정성

- COM 포트 드롭다운을 열 때 장치 목록을 즉시 새로 고쳐, 실행 후 연결된
  USB-UART 장치를 앱 재시작 없이 선택할 수 있습니다.
- Terminal/HEX 표시 모드와 HEX timeout을 변경할 때 활성 COM 연결을
  끊지 않고 수신 파이프라인에 즉시 적용합니다.

## HEX 패킷 그룹 기본값

- 새 프로필의 HEX timeout 기본값을 baud와 프레임 형식 권장값에 2ms를
  더한 값으로 자동 계산합니다.
- 사용자가 `Set`으로 지정한 값은 사용자 설정으로 보존하고, 자동 모드의
  값은 baud, data bits, parity, stop bits 변경을 따라갑니다.
- 손상되거나 범위를 벗어난 timeout은 현재 프로필의 통신 설정을 기준으로
  안전하게 복구합니다.

## 검색과 화면 로그

- 검색 대상에서 타임스탬프와 `RX <`, `TX >`, `MARK >`, `SYS` 방향
  메타데이터를 제외하고 실제 payload만 검색합니다.
- 표시 로그와 xterm scrollback이 하나의 공용 줄 제한을 사용하도록 설정을
  단순화했습니다.
- 각 논리 로그 줄에 내부 식별자를 부여해 긴 줄이 래핑되거나 검색 결과가
  stale 상태인 동안 새 로그가 추가되어도 같은 줄로 정확히 이동합니다.
- 내부 검색 식별자는 화면 복사와 저장 로그에는 포함되지 않습니다.

## 검증

- WinUI 테스트 164개와 Core 테스트 35개를 통과했습니다.
- xterm 인라인 JavaScript 문법 검사와 `git diff --check`를 통과했습니다.
- 긴 래핑 로그와 stale 검색 결과 조합을 수동 체크리스트에 추가했습니다.

## 프리릴리스 안내

- 실제 USB-UART 어댑터별 장시간 검증이 남아 있어 프리릴리스로 제공합니다.
- 설치 파일은 코드 서명이 없어 Windows SmartScreen에서 알 수 없는 게시자
  경고가 표시될 수 있습니다.
