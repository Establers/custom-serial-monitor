# Serial Monitor v1.1.5

출시일: 2026-07-19

## 로그 설정과 저장 안정성

- Settings 안에 있던 로그 설정을 별도 Log 탭으로 분리했습니다.
- 날짜가 바뀔 때 새 로그 파일을 만드는 일 단위 로테이션을 제거했습니다.
- 크기 로테이션 값을 MB 단위로 통일하고 비어 있거나 잘못된 값은
  10 MB로 복구합니다.
- 화면에 표시되는 MB 값과 실제 파일 로테이션 기준이 어긋나지 않도록
  소수 MB 값을 정규화합니다.
- 로그가 꺼진 뒤에도 Open 등 파일 관련 작업을 사용할 수 있습니다.

## 검색과 로그 규칙

- Search 자동 갱신을 제거하고 항상 수동 갱신으로 동작하게 했습니다.
- 로그 규칙은 항상 `New logs only`로 적용됩니다.
- 화면 재구성이나 표시 모드 변경 뒤에도 과거 로그에 새 필터·하이라이트가
  소급 적용되지 않도록 규칙 스냅샷을 보존합니다.

## Sequence와 HEX 송신

- Sequence별 Terminal/HEX 선택과 Enabled 옵션을 제거했습니다.
- Sequence 실행을 시작할 때의 메인 TX 모드를 사용하며, 실행 도중에는
  해당 모드를 유지합니다.
- HEX 모드에서는 모든 Sequence 스텝을 실행 전에 검증하여 일부 명령만
  전송된 뒤 실패하는 상황을 방지합니다.
- 공용 HEX 파서를 분리하고 유효·오류 입력 회귀 테스트를 추가했습니다.

## 브리지와 Release 안전성

- 브리지 포트 새로고침 뒤 선택이 사라져 시작할 수 없던 문제를 수정했습니다.
- Release 빌드에서는 MOCK Test 포트 설정을 강제로 끄고 Test 탭을 제외합니다.
- Sequence 실행 중 편집을 비활성화하여 실행 스냅샷과 UI 상태 충돌을 줄였습니다.

## 검증

- Core 테스트 35개와 WinUI 테스트 145개, 총 180개를 Debug와 Release에서
  각각 통과했습니다.
- Portable/Installer 패키지와 SHA-256 체크섬을 생성해 검증합니다.
- `git diff --check`를 통과했습니다.

## 프리릴리스 안내

- 실제 USB-UART 어댑터별 장시간 검증이 남아 있어 프리릴리스로 제공합니다.
- 설치 파일은 코드 서명이 없어 Windows SmartScreen에서 알 수 없는 게시자
  경고가 표시될 수 있습니다.
